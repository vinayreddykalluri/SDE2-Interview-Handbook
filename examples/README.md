# Runnable Examples

This directory contains executable source code only. Long-form explanations belong in `docs/` and should link here when code supports the lesson.

## Available Implementation

- [Java examples](java/README.md)
- `codingfoundations/<topic>/` contains semantic packages aligned with the 19 coding-foundation modules.
- `problemsolving/` contains enhanced examples for the backend programming-problem-solving track.
- `src/test/` contains smoke coverage across representative examples.

## Contributor Rule

Keep each example focused on one interview concept, compile it independently, include boundary behavior, and avoid framework setup unless the framework itself is the lesson. Run `make validate-code` after changing Java.
