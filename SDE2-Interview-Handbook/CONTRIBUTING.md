# Contributing

Thank you for contributing to the handbook.

## Style

- Use the chapter template in `docs/chapter-template.md`.
- Keep prose practical and interview-oriented.
- Keep diagrams in Mermaid and provide working Java snippets.

## Review process

1. Add or edit content.
2. Run `make validate`.
3. Build artifacts with `make build-all` and verify outputs.
4. Open a pull request with a short summary and review notes.
