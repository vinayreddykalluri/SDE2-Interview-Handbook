# Support

## Learning and usage questions

Search the [published handbook](https://vinayreddykalluri.github.io/SDE2-Interview-Handbook/) first. If the answer is missing or unclear, open a content-improvement issue with the chapter, question, and expected clarification.

## Build and tooling problems

Before opening an issue:

1. Confirm Python 3.11+, JDK 17+, Pandoc, and XeLaTeX are installed.
2. Run `make install`.
3. Run the failing command again and retain the complete error output.
4. Check existing issues for the same environment and failure.

Open an issue with the operating system, tool versions, command, smallest reproducible input, expected result, and actual result.
