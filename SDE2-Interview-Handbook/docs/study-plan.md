# Study Plan

## 12-week SDE-2 track

### Week 1-3
- Java execution model, JVM architecture, memory model, GC.

### Week 4-5
- Java core syntax, flow control, methods, arrays, strings.

### Week 6-8
- DSA foundations: recursion, hashing, two pointers, binary search, stacks/queues.

### Week 9-10
- Collections, graph/DP fundamentals, timed mock interviews.

### Week 11-12
- Mixed rounds simulation and full revision.

## Weekly rhythm

- 5 days focused study.
- 3 sessions per day:
  - 40 minutes theory
  - 50 minutes coding
  - 30 minutes revision

## Review rules

- Revisit weak chapters every 48 hours.
- Every Sunday: complete 5 interview questions from chapter 16.
