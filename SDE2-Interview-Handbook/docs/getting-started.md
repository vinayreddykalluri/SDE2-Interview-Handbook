# Getting Started

This handbook is Markdown-first and designed for both reading online and printing.

## Quick setup

```bash
make install
make serve
```

## Content workflow

Use a chapter-first approach:

- Read concept sections first.
- Review execution flow diagrams and code.
- Confirm edge cases from interview exercises.
- Revisit revision checklists and summary notes.

## Output expectations

- `output/pdf/Volume-01-Java-Fundamentals.pdf`
- `output/combined/SDE2-Interview-Handbook.pdf`
- `output/docx/Volume-01-Java-Fundamentals.docx`
- `output/combined/SDE2-Interview-Handbook.docx`
